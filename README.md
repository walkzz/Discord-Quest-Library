# Orb Automator

A lightweight application that allows you to launch an executable for any game. It fools Discord into thinking the game is active, letting you earn Discord orbs or customize your active status without wasting CPU or RAM on running the actual game.

## Features
* **Lightweight Background Process:** Completely resource-free.
* **Standalone Output:** Once launched, it stays in the background, allowing you to do other tasks without the need to spend 15 minutes stuck in-game.
* **UI Design:** Comes with a clean user interface, making it easy to select which game you want to start a quest for.

## Prerequisites
Just download the latest version from the **Releases** section on the GitHub page. It is named `Quest Launcher v1.0.0`.

## Step-by-Step Setup

1. Download the release version: go to the Releases page, navigate to **Assets** at the bottom, and download `Quest-Launcher-v1.0.0.zip`.
2. Extract the files to your desktop or anywhere you want on your computer.
3. Once extracted, open the folder and double-click on `Quest Launcher.exe`.
4. A Windows security screen may pop up saying *"Windows protected your PC"* or *"Microsoft Defender SmartScreen prevented an unrecognized app from starting"*. Click on **More info** and then click **Run anyway**.

> **IMPORTANT NOTE REGARDING DISCORD QUESTS:**
> Discord's API for orb quests is not public, so new quests won't automatically appear inside the application. I will constantly update the application, especially the `discord-executables.txt` file. This file is essential because all the games are stored there, including their specific titles and executables. 
> 
> To work around this problem, I will update the `discord-executables.txt` file regularly whenever new quests appear. Please contact me anytime you want a new game to be added to the list!

## FAQ

**1. Will I get banned for using this service?**
Technically, no. You might get rate-limited, which lasts for one or two days, but as of recently, this behavior isn't showing up as much.

**2. How many times can I use this program to earn orbs in a day?**
It is advised not to go over 3 times per day. Discord will rate-limit you and you will have to wait a day or two for it to reset.

**3. Can I use my earned orbs to gift my friends, especially items from their wishlist?**
No, you cannot gift orbs to your friends. However, you can share this application with them so they can earn their own!

**4. Is this service free?**
The service is completely open-source and free. I do not need your well-earned money.
